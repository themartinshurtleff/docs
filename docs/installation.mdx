# Installation

TradeNet Terminal is currently in closed beta. You need waitlist approval before you can download it. Once approved, you will receive a download link via email.

## System Requirements

| Requirement | Minimum |
|---|---|
| **OS** | Windows 10+ or macOS 12+ |
| **RAM** | 8 GB |
| **GPU** | Vulkan-compatible (recommended) |
| **Network** | Stable internet connection |

A dedicated GPU with Vulkan support is strongly recommended. The terminal uses GPU-accelerated rendering for charts and heatmaps. It will fall back to software rendering without one, but performance will suffer.

## Windows

1. Open the download link from your approval email.
2. Run the `.msi` installer.
3. Follow the prompts. Default install location is `C:\Program Files\TradeNet Terminal`.
4. Launch from the Start Menu or desktop shortcut.

Windows Defender or your antivirus may flag the installer. This is normal for unsigned beta software. Allow the installation to proceed.

## macOS

1. Open the download link from your approval email.
2. Download the `.dmg` file.
3. Open the `.dmg` and drag TradeNet Terminal into your Applications folder.
4. On first launch, macOS will block it. Go to **System Settings > Privacy & Security** and click **Open Anyway**.

Apple Silicon (M1/M2/M3/M4) and Intel Macs are both supported. The binary is universal.

## Updates

The terminal checks for updates on launch. When a new build is available, you will see a prompt to download and install it. Updates are incremental and fast. No need to reinstall.
